      均值为 1，0，方差均为 1，相关系数为 0 → 两变量独立

      P(XY−Y<0)=P(Y(X−1)<0)=1/2 \mathbb{P}(XY - Y < 0) = \mathbb{P}\big(Y(X-1)<0\big) = 1/2 P(XY−Y<0)=P(Y(X−1)<0)=1/2

      显然就是均值的两边嘛一个大一个小，一个小一个大，刚好一半
