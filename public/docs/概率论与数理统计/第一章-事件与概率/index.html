      第一章 事件与概率

      教学目的:1) 掌握随机事件的概念和相关运算.2)了解概率的不同定义,掌握古典概型的基本计算.3) 掌握条件概率的概念,熟练运用全概率公式和Bayes公式.4) 掌握事件独立的概念和有关运算.\begin{aligned}&\text{教学目的:}\\&1)\text{ 掌握随机事件的概念和相关运算}.\\&2)\text{了解概率的不同定义,掌握古典概型的基本计算}.\\&3)\text{ 掌握条件概率的概念,熟练运用全概率公式和Bayes公式}.\\&4)\text{ 掌握事件独立的概念和有关运算}.\end{aligned}​教学目的:1) 掌握随机事件的概念和相关运算.2)了解概率的不同定义,掌握古典概型的基本计算.3) 掌握条件概率的概念,熟练运用全概率公式和Bayes公式.4) 掌握事件独立的概念和有关运算.​

      1. 随机事件概念和运算

      • 随机试验：在相同条件下重复进行的试验，结果具有不确定性。
      • 随机事件：由若干基本事件组成的集合，如 {点数为 1}、{出现两次} 等。
      • 样本空间（Ω\OmegaΩ）：所有可能的基本事件的集合。
        • 示例：Ω={1,2,3,4,5,6}\Omega = \{1,2,3,4,5,6\}Ω={1,2,3,4,5,6}（掷骰子）或 Ω={x∣0≤x<T}\Omega = \{x | 0 \leq x < T\}Ω={x∣0≤x<T}（时间区间）。
      • 必然事件：一定发生的事件，即 Ω\OmegaΩ。
      • 不可能事件：一定不发生的事件，即 ∅\varnothing∅。

      事件的运算

      • 子事件：A⊂BA \subset BA⊂B，表示 AAA 发生时 BBB 一定发生。
      • 事件的和（或门）：A∪BA \cup BA∪B 或 A+BA + BA+B，表示至少一个事件发生。
      • 事件的积（与门）：A∩BA \cap BA∩B 或 ABABAB，表示两个事件同时发生。
      • 对立事件（非门）：AcA^cAc 或 Aˉ\bar{A}Aˉ，表示 AAA 不发生。
      • 事件的差：A−B=ABcA - B = AB^cA−B=ABc，表示 AAA 发生但 BBB 不发生。

      De Morgan 对偶法则

      A∪B‾=Aˉ∩BˉA∩B‾=Aˉ∪Bˉ \begin{aligned} \overline{A \cup B} &= \bar{A} \cap \bar{B} \\ \overline{A \cap B} &= \bar{A} \cup \bar{B} \end{aligned} A∪BA∩B​=Aˉ∩Bˉ=Aˉ∪Bˉ​

      对于 nnn 个事件：

      ⋃i=1nAi‾=⋂i=1nAˉi⋂i=1nAi‾=⋃i=1nAˉi \begin{aligned} \overline{\bigcup_{i=1}^n A_i} &= \bigcap_{i=1}^n \bar{A}_i \\ \overline{\bigcap_{i=1}^n A_i} &= \bigcup_{i=1}^n \bar{A}_i \end{aligned} i=1⋃n​Ai​​i=1⋂n​Ai​​​=i=1⋂n​Aˉi​=i=1⋃n​Aˉi​​

      2. 概率定义

      • 概率：事件发生的可能性，范围在 0 到 1 之间。
      • 古典概型：
        • 有限性：只有有限个结果。
        • 等可能性：每个基本事件发生的概率相同。
        • 公式：若 AAA 包含 mmm 个基本事件，则
      P(A)=mn=#(A)#(Ω) P(A)=\frac mn = \frac{\#(A)}{\#(\Omega)} P(A)=nm​=#(Ω)#(A)​

      其中 nnn 是样本空间的大小。

      概率加法公式

      • 若 AAA 与 BBB 不相容（互斥），则：
        P(A∪B)=P(A)+P(B) P(A \cup B) = P(A) + P(B) P(A∪B)=P(A)+P(B) P(⋃i=1∞Ai)=∑i=1∞P(Ai) P(\bigcup\limits_{i=1}^\infty A_i)=\sum\limits_{i=1}^\infty P(A_i) P(i=1⋃∞​Ai​)=i=1∑∞​P(Ai​)

      3. 各种概率公式

      条件概率

      P(A∣B)=P(AB)P(B) P(A|B) = \frac{P(AB)}{P(B)} P(A∣B)=P(B)P(AB)​
      • 表示在 BBB 发生的前提下，AAA 发生的概率。
        P(A∣B)=P(AB)P(B)⇒P(AB)=P(A∣B)P(B) P(A|B)=\frac{P(AB)}{P(B)}\Rightarrow P(AB)=P(A|B)P(B) P(A∣B)=P(B)P(AB)​⇒P(AB)=P(A∣B)P(B) P(A1A2⋯An)=P(A1)P(A2∣A1)⋯P(An∣A1⋯An−1) P(A_1A_2\cdots A_n)=P(A_1)P(A_2|A_1)\cdots P(A_n|A_1\cdots A_{n-1}) P(A1​A2​⋯An​)=P(A1​)P(A2​∣A1​)⋯P(An​∣A1​⋯An−1​)

      全概率公式

      设 B1,B2,…,BnB_1, B_2, \dots, B_nB1​,B2​,…,Bn​ 是样本空间的一个分割（即 ⋃i=1nBi=Ω\bigcup_{i=1}^n B_i = \Omega⋃i=1n​Bi​=Ω 且 BiB_iBi​ 两两互斥），则：

      P(A)=∑i=1nP(A∣Bi)P(Bi) P(A) = \sum_{i=1}^n P(A|B_i)P(B_i) P(A)=i=1∑n​P(A∣Bi​)P(Bi​)

      Bayes 公式

      P(Bi∣A)=P(A∣Bi)P(Bi)∑j=1nP(A∣Bj)P(Bj) P(B_i|A) = \frac{P(A|B_i)P(B_i)}{\sum_{j=1}^n P(A|B_j)P(B_j)} P(Bi​∣A)=∑j=1n​P(A∣Bj​)P(Bj​)P(A∣Bi​)P(Bi​)​
      • 用于从结果反推原因的概率。

      独立性

      • 若 P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B)，则称事件 AAA 和 BBB 相互独立。
      • 更直观的定义是 P(A∣B)=P(A)P(A|B) = P(A)P(A∣B)=P(A)，即 BBB 的发生不影响 AAA 的概率。

      P(A~1A~2⋯A~n)=P(A~1)P(A~2)⋯P(A~n) P(\tilde{A}_1\tilde{A}_2\cdots\tilde{A}_n)=P(\tilde{A}_1)P(\tilde{A}_2)\cdots P(\tilde{A}_n) P(A~1​A~2​⋯A~n​)=P(A~1​)P(A~2​)⋯P(A~n​)
